Using QuickCheck with Hspec

You can use arbitrary QuickCheck properties with Hspec, but they must be wrapped with QuickCheck's property function. Here is an example:

describe "read" $ do
  it "is inverse to show" $ property $
    \x -> (read . show) x == (x :: Int)
Example code:
-- file Spec.hs
import Test.Hspec
import Test.QuickCheck

main :: IO ()
main = hspec $ do
  describe "read" $ do
    context "when used with integers" $ do
      it "is inverse to show" $ property $
        \x -> (read . show) x == (x :: Int)

$ runhaskell Spec.hs
